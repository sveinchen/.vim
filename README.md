# My Vim Config Files! 

## Installation

1. Clone and install this repo:

```
git clone git://github.com/sveinchen/.vim.git ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
```

2. Init and install submodules:

```
cd ~/.vim
git submodule init
git submodule update
```

3. Install Plugins:

```
Launch `vim` and run `:PluginInstall`
```
